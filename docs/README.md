# Project Documentation Hub

This folder tracks project direction, progress, and implementation details for the Shape Seeds UX Portfolio.

## Documents

- `01-project-scope.md` - Scope, outcomes, in-scope/out-of-scope, and milestones.
- `02-project-aims.md` - Strategic aims, measurable targets, and success criteria.
- `03-technical-overview.md` - Architecture, stack, patterns, and technical constraints.
- `04-change-log.md` - Dated record of major updates and their impact.
- `05-decision-log.md` - Architectural/product decisions with rationale and trade-offs.

## Update Rhythm

- **Before starting a major change:** update `01-project-scope.md` if scope shifts.
- **When priorities change:** update `02-project-aims.md`.
- **When implementation changes:** update `03-technical-overview.md`.
- **After shipping work:** add an entry in `04-change-log.md`.
- **When making non-obvious calls:** add a record in `05-decision-log.md`.

## Ownership

- Product direction: Project owner
- Design and UX quality: Project owner
- Technical implementation: Development owner
